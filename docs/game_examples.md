### Roguelike 融合

#### 1. 《暗黑地牢》(Darkest Dungeon)
**融合类型**: Roguelike + 策略管理 + 心理恐怖

**核心融合**:
- Roguelike 的永久死亡 → 英雄压力系统
- 策略管理的资源分配 → 英雄状态管理
- 心理恐怖 → 压力值影响行为

**成功要素**:
- 压力系统创新（核心亮点）

## 📊 融合模式分析

### 常见融合模式

| 模式 | 案例数 | 成功率 | 代表作品 |
|------|-------|--------|---------|
| **Roguelike + X** | 8 | 高 | 哈迪斯、暗黑地牢 |
| **RPG + X** | 10 | 高 | 巫师 3、极乐迪斯科 |
| **模拟经营 + X** | 6 | 高 | 环世界、冰汽时代 |
| **策略 + X** | 5 | 中 | 十字军之王 |
| **动作 + X** | 7 | 高 | 只狼、对马岛 |
| **解谜 + X** | 4 | 中 | 传送门、Baba Is You |
| **恐怖 + X** | 4 | 中 | 生化危机、异形 |
| **竞速 + X** | 3 | 中 | 火箭联盟 |

### 成功要素总结

1. **核心循环清晰** - 融合后玩法不混乱
2. **创新点突出** - 至少一个亮点
3. **目标受众明确** - 知道为谁设计
4. **执行质量高** - 美术/音乐/优化到位
5. **市场时机好** - 填补市场空白

## 💡 使用建议

### 学习融合思路

1. **选择类似案例** - 找与你创意相近的成功作品
2. **分析融合点** - 看它们如何整合两种类型
3. **提取设计原则** - 总结可复用的经验
4. **应用到你的创意** - 借鉴但不抄袭

### 验证创意质量

用案例库对比你的创意：
- 创新性：有没有类似作品？
- 可行性：类似作品成功了吗？
- 市场：类似作品销量如何？
